# 2026-02-02：CLI Only 架构（MCP 已移除）

## 背景

本项目已收敛为 CLI Only：对外入口为独立 CLI 工程；内部复用既有 Services + Tools 作为能力库。

## 模块划分

- `Windows.Agent.Cli`：对外入口、命令解析、统一 JSON 输出（stdout）、日志（stderr）。
- `Windows.Agent`：能力库（Services + Tools），不包含可执行入口，也不包含 MCP 元数据。

## 调用链

CLI -> Tool（参数保持一致）-> Service（实际执行）

## 输出与日志约定

- CLI 结果：stdout 输出（默认 JSON，可 `--pretty`）。
- 诊断日志：写入 stderr（避免污染 stdout）。

## 限制

- 由于调用的是同一套 Windows 桌面自动化实现，CLI 仍然需要运行在“被操作的那台 Windows 机器”上。
